# **📐 Math Learning Studio**

**小学校の算数授業のために作られた、図形描画・敷き詰め学習Webアプリ**

Math Learning Studioは、プログラミングやインストールの知識がなくても、ブラウザさえあればすぐに使える図形学習ツールです。

「図形の敷き詰め」「合同な図形」「拡大・縮小」「面積」などの単元で活用できます。

## **✨ 特徴 (Features)**

* **⚡ 完全インストール不要**: index.html 1ファイルだけで動作します。ネット環境がない場所でも、ファイルをUSBで配るだけで使えます。  
* **🟦 方眼紙スナップ**: 図形がピタッとマス目に吸着。几帳面な図形作成や敷き詰め学習に最適です。  
* **✏️ 頂点編集モード**: 四角形や三角形を自由に変形し、子どもたちの想像力を引き出します。  
* **📏 本格的な定規・分度器**: 半透明で下の図形が見える、実用的な計測ツールを搭載。  
* **💾 自動保存 & やり直し**: ブラウザを閉じても続きから始められます。失敗しても「戻る」ボタンで安心。  
* **🏫 GIGAスクール対応**: 全ての漢字にふりがな（ルビ）付き。タブレット操作に最適化されたUI。

## **🛠 機能一覧**

| 機能 | 説明 |
| :---- | :---- |
| **図形追加** | 四角、三角、円、多角形（正多角形作成も可能） |
| **サイズで作る** | 「横3マス、縦4マス」のように指定して正確な図形を作成 |
| **形を変える** | 頂点をドラッグして、自由な形の多角形に変形 |
| **定規・分度器** | 半透明のデザイン。回転・移動して長さや角度を計測可能 |
| **画像保存** | 作った作品をPNG画像としてダウンロード |
| **方眼/角度スナップ** | マス目や15度刻みの角度に自動で合わせる機能（ON/OFF可） |

## **💻 技術スタック (Tech Stack)**

このアプリはメンテナンス性を考慮し、外部サーバーや複雑なビルドプロセスを排除した「シングルファイル・アプリケーション」として設計されています。
